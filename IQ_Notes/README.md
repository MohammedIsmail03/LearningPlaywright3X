# IQ_Notes Template

Use this prompt to create new concept explanation files in this folder.

## Prompt Template

```
Explain <concept_name> with the 01_chapter_Basics/01_HelloWorld.js example.
Create a markdown file in IQ_Notes/ named <Concept_Name>_IQ.md using the
same table-and-pipeline format as Source_Code_ByteCODE_Binary_IQ.md.
```

## Examples

```
Explain compilation vs interpretation with the 01_chapter_Basics/01_HelloWorld.js example.
Create a markdown file in IQ_Notes/ named Compilation_vs_Interpretation_IQ.md using the
same table-and-pipeline format as Source_Code_ByteCODE_Binary_IQ.md.
```

## Prompt Template (Alternate)

```
Explain [YOUR CONCEPT] in a table view with clear comparison columns. Use a simple
code/file example to walk through each layer. Save it as a `.md` file inside the
IQ_Notes folder. Follow the same visual style as Source_Code_ByteCODE_Binary_IQ.md —
breakdown table, example walkthrough, pipeline diagram, and TL;DR.
```
